CR sim for decentralized sequencers

For L2s with sequencer committees or fully decentralized sequencers, this calculator shows how long a censored user would have to wait, and how much they would have to stake, to include their transaction with a given probability. Config defaults model a static sequencer set with 50% censoring, a 99% inclusion target, and the escape hatch enabled. The hardcoded quorum for committees is 2/3.
